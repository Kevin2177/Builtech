Creacion de Paginas Web

Funciones

La pagina contara con diferentes funciones las cuales seran propuestas por nuestros clientes, entre algunas funciones basicas que se podran implementar seran ( Croquis, Ubicacion enlazada con Google Maps, Quejas y Sugerencias del negocio, etc.)

Beneficios

Uno de los beneficos de las paginas web es el poder mostrar la ubicaion tu negocio en una interfaz grafica la cual ayudara a orientar más a tus clientes. Tambien se te dara elegir entre los diferentes tipos de ilustraciones web las cuales se adaptaran a tu negocio, así como una mejor visualización de los productos que manejes o realices. Dando una gran publicidad con tus clientes y nuevos clientes que le interesen.

Modelos

Los modelos de las paginas web es el tipo de estructura que nuestros clientes deseen, asi como los tipos de diseños que este desee, para que asi tengan el mejor modelo para ellos y sus clientes.

Ilustraciones

Las ilustraciones tienes varias caracteristicas las cuales, haran mas llamativa la pagina web estas se adaptaran a la interfaz de la tienda y a las desiciones que tome el dueño de la pagina, lo cual es fundamental para poder tomar las diferentes caracteristicas de este entre las cuales se encuentra las fotografias comunes, editadas, animadas, etc.
